# 🌿 Greenden – Plant Store Website

Greenden is a modern and responsive plant store website built using **HTML, Tailwind CSS, and JavaScript**.  
This project showcases a clean UI design and interactive features such as product search and responsive navigation.

## 🚀 Features

- 🌱 Responsive design for mobile, tablet, and desktop
- 🔍 Real-time product search functionality
- 📱 Mobile-friendly side navigation menu
- 🛒 Product cards with pricing and ratings
- 🎨 Modern UI using Tailwind CSS
- ⚡ Smooth hover and transition effects

## 🛠️ Technologies Used

- **HTML5**
- **Tailwind CSS**
- **JavaScript (Vanilla JS)**
